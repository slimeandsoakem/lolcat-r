# lolcat in rust
Inspired by https://github.com/jaseg/lolcat - just written in Rust so the hipsters can get onboard with it

## Usage
```shell
lolcat-r -h <horizontal frequency (0,1:default 0.23)> -v <vertical frequency (0,1: default 0.1)
```

## Building and installing
1. Install a rust toolchain
2. Clone this repo
3. Run the make commands
```shell
make build test install 
```
